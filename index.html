Embedded monitor

Sniffy embedds a small icon to your application allowing you to get the monitoring data for the current page (like SQL queries, REST calls, e.t.c.)

Easy setup

Installing sniffy is as easy as adding a Servlt Filter to your web.xml and changing your database connection URL

Unit testing

Sniffy provides a framework for validating the executed database queries. It is excellent for finding N+1 issues
